## [0.1.0] - 2024-11-29

### Added
- Initial release
- Basic neural network implementation
- Support for multiple activation functions (sigmoid, tanh, relu, leaky_relu)
- Support for multiple loss functions (bce, ce, mse)
- Xavier and He weight initialization
- Forward and backward propagation
- Training loop
- Examples for XOR, AND gate, and regression problems
